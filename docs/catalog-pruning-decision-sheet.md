# Catalog Pruning Decision Sheet

Working curation sheet for the current site catalog.

This does **not** apply deletions by itself. It is the agreed shortlist for future cleanup work, based on:

- current site data in `backend/data/posts.json`
- curation intent in `docs/site-memory.md`
- later Suno generations and screenshot review

## Principles

- Delete old Donna/archive filler aggressively.
- Keep the real Fractureverse, Eldoria, and trans/identity spine.
- Preserve the anime/princess lane selectively instead of wiping it out.
- Do not delete version families that still need a final winner.
- Do not delete `we-were-never-meant-to-survive` until a trusted duet replacement exists.
- Screenshot matching uses song or family title presence, not exact site slug labels.
- If a non-Donna song family never appeared in the screenshot set, it now falls into `Delete Now`.
